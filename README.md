De tutorials jag använde var exempelvis geeks for geeks flexbox och vissa små grejer från andra tutorials.

Jag tycker att många av tutorialsen jag provade var bra dock så finns det vissa tutorials som inte hade den informationen om flexbox som man kanske sökte sig, vissa av dem hade inget om media quaries och små scripter som finns i denna htmlen.

Jag tycker de gick bra att använda dessa tutorials samt använde jag kunskaper sedan tidigare lektioner för att skapa denna övnings websidan. 

jag skulle verkligen kunna göra denna hemsida bättre om jag hade inte suttit fast på media quariesen så länge.

det jag tyckte var bra med till exempel mozilla var att det fanns mer utförlig information om vad sakerna gjorde, och i exempelvis w3schools fanns det en knapp på vardera function som körde den och visade hur det såg ut, i den editorn kunde du ändra och lägga till för att se vad som hände vilket jag tyckte var bra.

jag skulle rekomendera antingen mozilla eller w3schools eller båda eftersom de är inte så krångliga att förstå och har mycket information om just flexbox och html functioner.

som sakt det som var mindre bra var att många inte hade media quaries som en standard del i flexbox delarna vilket är lite synd eftersom media quaries är så jävulskt bra för att kunna fixa websidan för telefoner.

jag skulle ge den en overall score av 8.7/10 eftersom jag visste vart jag kunde hitta den mesta informationen på hemsidorna eftersom de hade gjort det lätt att veta vart saker fanns.
